# Proposal: enhance-ux

## Context
当前版本在生成 AI 健身计划时存在较长的黑盒等待期（点击提交后页面无明确反馈），且生成的报告深度有待加强。

## Goals
- **引入多阶段 UX 体验**: 点击提交后进入带有 3 分钟进度模拟和知识轮播的“加载动画页”，缓解用户焦虑。
- **结构化结果页**: 生成完毕后自动跳转至独立的结果展示页，保持主表单页的纯洁性。
- **深度化内容输出**: 强化 AI 提示词，让生成的评估报告和健身计划在组数、次数、饮食结构等方面更加详尽和专业。

## Scope
- 核心修改 `app.py`（Streamlit 应用逻辑）。
- 引入 Python 多线程 (`threading`) 管理模型异步调用。
- 更新 `project.md` 的规范描述。
